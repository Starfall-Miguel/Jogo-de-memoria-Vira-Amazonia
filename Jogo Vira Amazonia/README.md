# Jogo "Vira Amazônia" — Instruções para rodar

Este projeto é um jogo da memória implementado com HTML, CSS e JavaScript. Este README traz os requisitos e instruções para rodar o jogo localmente — apenas este arquivo será criado/alterado.

**Requisitos mínimos**
- Navegador moderno (Chrome, Edge, Firefox)
- (Opcional) Python 3.x ou Node.js para iniciar um servidor HTTP local

**Como executar (opções)**

- Abrir direto (rápido):
  - Abra `index.html` com um duplo-clique no Explorador de Arquivos.


**Observações rápidas (problemas conhecidos no código)**
- A pasta de imagens no projeto é `Imagens/` (com I maiúsculo). Se o código usa `imagens/` em minúsculas, em sistemas case-sensitive isso falhará.
- Dependendo de quão rápido as cartas forem pressionadas, o usuário pode acabar invertendo o grid de cartas por um breve momento, então voltará ao normal.
